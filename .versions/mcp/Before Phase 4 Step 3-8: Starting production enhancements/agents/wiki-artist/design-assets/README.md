# Design Assets Library

This directory contains reusable design assets for the Wiki-Artist agent.

## Contents

### Color Palettes
- GitHub default theme colors
- Dark/light mode compatible palettes
- Accessibility-compliant color schemes
- WCAG 2.1 AA compliant contrast ratios

### Icon Sets
- GitHub Octicons (official)
- Font Awesome Free subset
- Material Design Icons
- Simple Icons (brand logos)
- Feather Icons

### Badge Templates
- Status indicators
- Version badges
- Build/test status
- Technology stack
- License information
- Coverage metrics

### Diagram Templates
- Flowchart patterns
- Sequence diagram templates
- Architecture diagram layouts
- Gantt chart examples
- Pie chart configurations

### Layout Patterns
- Hero sections
- Feature grids
- Comparison tables
- Card layouts
- Collapsible sections
- Callout boxes

## Usage

All assets are free and open-source. Use them to enhance visual presentation while preserving content integrity.

## Updates

This library is continuously updated based on:
- Design trend analysis
- User engagement metrics
- Accessibility improvements
- Community feedback
- MCP container learning data
