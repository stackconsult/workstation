    ğŸ’» Installation & Setup

    Local Development

    1. Clone the repository:
    git clone https://github.com/creditXcredit/workstation.git
    cd workstation
    1. Install dependencies:
    npm install
    1. Configure environment:
    cp .env.example .env
    # Edit .env with your settings

    Security Note: Generate a secure JWT secret:

    node -e "console.log(require('crypto').randomBytes(32).toString('hex'))"
    1. Start development server:
    npm run dev
    1. Build for production:
    npm run build
    npm start

    Docker Deployment

    # Build image
    docker build -t workstation .

    # Run container
    docker run -p 3000:3000 -e JWT_SECRET=your_secret workstation

    Create a Workflow

    # Get authentication token
    TOKEN=$(curl -s http://localhost:3000/auth/demo-token | jq -r '.token')

    # Create workflow
    curl -X POST http://localhost:3000/api/v2/workflows \
      -H "Authorization: Bearer $TOKEN" \
      -H "Content-Type: application/json" \
      -d '{
        "name": "Google Search",
        "description": "Search and screenshot",
        "definition": {
          "tasks": [
            {
              "name": "navigate",
              "agent_type": "browser",
              "action": "navigate",
              "parameters": {"url": "https://google.com"}
            },
            {
              "name": "screenshot",
              "agent_type": "browser",
              "action": "screenshot",
              "parameters": {"fullPage": true}
            }
          ]
        }
      }'
