## ğŸ“ What Can You Do?

The Browser Agent lets you automate any web task:

1. **Web Scraping** - Extract data from websites
2. **Form Automation** - Fill forms automatically
3. **E2E Testing** - Automate browser testing
4. **Monitoring** - Check websites regularly
5. **Screenshots** - Capture page images
6. **Data Collection** - Gather information at scale
7. **Custom Scripts** - Run JavaScript in browsers

**7 Powerful Actions:**
- `navigate` - Go to URLs
- `click` - Click elements
- `type` - Fill inputs
- `getText` - Extract text
- `screenshot` - Capture images
- `getContent` - Get HTML
- `evaluate` - Run JavaScript
